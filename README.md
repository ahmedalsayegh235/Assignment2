# University of Bahrain - Students Enrollment by Nationality

## Description

This project involves fetching and displaying data on the enrollment of students at the University of Bahrain (UOB) based on their nationality. The data is retrieved from the Bahrain Open Data Portal API, specifically the dataset titled "University of Bahrain Students Enrollment by Nationality."

The application fetches this data and presents it in a user-friendly, responsive HTML table format. The table includes details such as:
- Year of enrollment
- Semester
- Program (Bachelor's)
- Nationality of students
- College
- Number of students enrolled

The project uses PHP to fetch and process data, and Pico CSS is used for styling to ensure the table is responsive and clean.

## Assignment Details

### Task 1: Data Retrieval
- Use the provided API to retrieve the data.
- Parse the JSON response from the API.
- Extract the necessary data and display it.

### Task 2: Data Visualization
- Display the extracted data in a table format using HTML.
- Make the table responsive using Pico CSS or a similar framework.
